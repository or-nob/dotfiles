# vim-configs

## Run the following one by one.
`git clone https://github.com/or-nob/vim-configs ~/.vim_config`<br/>
`cd ~/.vim_config`<br/>
`./install.sh`<br/>
`source ~/.vim_config`<br/>
`vim .`<br/>
Then, from vim command line:
`:PluginInstall`<br/>

Please, install clang-format for `vim-clang-format` plugin to work.
